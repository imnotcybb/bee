Bee API 调试页

固定请求 http://localhost:8080，请先启动应用。

学生 /student

查询学生：按学生 id 查询一条记录，返回 JSON。

删除学生：从数据库永久删除该 id 对应的学生（不可恢复，请谨慎）。

学生列表：返回当前库中全部学生；下方 id 为接口占位参数，服务端未使用。

新增学生：提交 JSON 创建学生，一般只需 name、age。

POST /student/add（JSON）

更新学生：按 JSON 中的 id 修改姓名、年龄等（使用 GET + body，部分浏览器可能不支持）。

GET /student/update（JSON 请求体，部分环境对 GET+body 支持有限）

响应将显示在这里

课程 /course

新增课程：提交 JSON 创建一门课，填写课程名、授课教师等。

POST /course/add

更新课程：按 JSON 中的 id 修改课程名称与教师。

POST /course/update

查询 / 删除课程：左侧填写课程 id；「查询」返回该课详情，「删除」从库中移除该课程（若仍有人选课，请视业务与数据库约束而定）。

响应将显示在这里

选课 / 退课（/studentCourse）

选课：指定学生与课程，在选课表中新增一条关联（重复选课会失败）。退课：取消该学生与该课程的关联。

选课：GET /studentCourse/addStudentCourse?studentId=&courseId=
退课：GET /studentCourse/drop?studentId=&courseId=（与 deleteStudentCourse 相同）

响应将显示在这里
